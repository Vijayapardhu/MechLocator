# MechLocator - Mechanic Shop Finder App

A comprehensive React Native mobile application that helps users find nearby mechanic shops, book services, and manage their automotive needs with a beautiful, modern UI.

## üöÄ Features

### Core Functionality
- **Location-based Shop Discovery**: Find mechanic shops near your current location
- **Advanced Filtering**: Filter shops by rating, distance, services, and availability
- **Interactive Maps**: View shops on an interactive map with real-time location
- **Shop Details**: Comprehensive shop information including services, hours, and contact details
- **Call Integration**: Direct calling to shops with one tap
- **Directions**: Get directions to shops using Google Maps integration

### User Features
- **User Authentication**: Secure login and registration system
- **Profile Management**: Update personal information and preferences
- **Service Booking**: Book appointments with mechanic shops
- **Review System**: Rate and review shops with comments and images
- **Push Notifications**: Get reminders for upcoming appointments
- **Booking History**: Track your service appointments

### Admin Features
- **Shop Management**: Add, edit, and delete mechanic shops
- **Working Hours**: Set and update shop operating hours
- **Service Management**: Manage available services for each shop
- **Review Management**: Monitor and manage user reviews
- **Analytics Dashboard**: View shop performance metrics

## üì¶ Installation

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn
- Expo CLI
- iOS Simulator or Android Emulator (for testing)

### Setup Instructions

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/mechlocator.git
   cd mechlocator
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm start
   ```

4. **Run on device/simulator**
   ```bash
   # For iOS
   npm run ios
   
   # For Android
   npm run android
   
   # For web
   npm run web
   ```
